# Quản lý thư viện sách 
Hệ thống quản lý thư viện viết bằng C# .NET 10

## Chức năng:
- Quản lý 3 loại sách: Tiểu thuyết, Sách giáo khoa, Sách tham khảo.
- Mượn/Trả sách và tự động tính phí phạt nếu quá hạn 14 ngày.
- Tìm kiếm sách theo tên hoặc ISBN.
- Thống kê số lượng sách trong kho.

## Cách chạy:
1. Mở bằng Visual Studio hoặc VS Code.
2. `dotnet run` để khởi chạy console.
